Plant health prediction
This dataset contains 1,200 records of plant health monitoring using sensor data. It includes environmental factors(temperature, humidity, light), soil conditions (moisture, pH, nutrients), and plant-specific parameters (chlorophyll, electrochemical signals). The target variable, Plant_Health_Status, categorizes plant conditions like "High Stress." It is suitable for machine learning models to predict plant health based on real-time sensor inputs, aiding in precision agriculture.
